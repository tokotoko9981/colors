## Introduction
出題される問題に対し答え、クリアを目指すゲームです。
協力プレイなど基本なんでもOKです。
また、PCでのプレイを想定していますが、想定しているだけでスマホ等でもプレイ可能だと思います。
(ブラウザは最新版のGoogle Chromeを推奨しますがこれも推奨しているだけです。)

## Rule
画面に出題される問題に対しキーパッドが用意されているのでそれを用いて入力してください。
×は消去、✓は確定を表します。
画面下部にはすでに到達した問題に移動することができるボタンがついています。
なんで戻れるかは察してください。
